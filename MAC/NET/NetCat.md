# Nmap

## 설명
넷캣은 TCP/IP 프로토콜을 사용하여 네트워크 연결 전반에 걸쳐 데이터를 읽고 쓰는 네트워킹 유틸리티입니다.

## 설치 영역
`/usr/bin/nc`  
`/opt/homebrew/bin/nc`  
`/opt/homebrew/bin/netcat`

## 사용처
- 네트워크 취약점 점검
- 서버 상태 파악

## 설치(접속) 방법
1. **Terminal 실행**
2. **실행 확인**
   ```bash
    nc --version
   ```
