# OWASP ZAP

## 설명
웹 애플리케이션 보안 테스트 도구로, 네트워크 트래픽 분석과 자동화된 취약점 점검을 지원합니다.

## 설치 영역
`/Applications/OWASP ZAP.app`

## 사용처
- 웹 애플리케이션 보안 점검
- 네트워크 트래픽 가로채기 및 분석
- 자동화된 취약점 스캔

## 설치(접속) 방법
1. **Homebrew로 설치:**
   ```bash
   brew install --cask owasp-zap
   ```
