# 해커 노트북 설정 가이드 (LINUX)

초보자 또는 다양한 해킹 도구를 효율적으로 탐색하려는 사용자를 위해 간단한 체크리스트를 통해 초기 설정 과정을 간소화하고 설정 관련 리소스를 줄이기 위해 제작되었습니다.

---

## 목적
이 가이드는 다음을 목표로 합니다:
- 초보 사용자가 필수 해킹 도구를 설치할 수 있도록 지원.
- 효율적인 설정을 위한 체크리스트 제공.
- 환경 수동 설정의 부담을 줄임.
