# BrowsingHistoryView
## 설명
웹 브라우저들의 방문 기록을 분석하고 확인할 수 있는 도구. 여러 브라우저의 히스토리를 동시에 분석하고 통합 조회할 수 있음.

## 설치 영역
`다운로드하여 실행하는 포터블 프로그램으로, 별도 설치 과정이 필요 없음`

## 사용처
* 디지털 포렌식
   * 사용자 웹 활동 분석
   * 악성 사이트 방문 기록 확인
   * 데이터 유출 경로 추적
* 시스템 관리
   * 사용자 인터넷 사용 모니터링
   * 웹 브라우저 사용 패턴 분석
   * 보안 정책 위반 조사
* 개인 사용
   * 방문했던 사이트 기록 찾기
   * 웹 활동 내역 정리
   * 브라우저 사용 기록 백업

## 설치(접속) 방법
1. **NirSoft 웹사이트에서 BrowsingHistoryView 다운로드**
2. **압축 파일 해제**
3. **BrowsingHistoryView.exe 실행**
