# Censys

## 설명
공개적으로 접근 가능한 네트워크 자산 및 취약점을 검색하는 도구.

## 설치 영역
`웹 기반 도구 - 별도의 설치 필요 없음`

## 사용처
- 취약점 탐지
- IoT 장치 분석
- 서버 노출 확인

## 설치(접속) 방법
1. **웹 브라우저를 열고 [Censys 공식 웹사이트](https://search.censys.io/)에 접속합니다.**
